REQUIREMENTS
============

Runtime
-------

The following optional packages enable extra runtime features.
They are not required to build or install the software.

* `p5-gtk2-trayicon`:
    Enables tray icon support.

* `p5-gtk2-notify`:
    Enables desktop notifications.

* `p5-gnome2-wnck`:
    Special layout in/around the titlebar of the focused window.

* `p5-gstreamer1`, `p5-glib-object-introspection`:
    Enables Gstreamer 1.x playback output.

* `mplayer`:
    Enables MPlayer playback output.

* `mpv`:
    Enables MPV playback output.

* `flac123`, `mpg123`/`mpg321`, `ogg123`:
    Enables FLAC, MP3/MP2, and OGA/OGG playback output.

* `grip`, `sound-juicer`, `xcfa`, or custom command:
    Enables CD ripping (via button).  Only one is needed;
    jukebox lets you choose which to use at runtime.

* `spek`
    Adds acoustic spectrum analyser (via context menu).
